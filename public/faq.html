1. How does the running data parsing feature work?

✅ Status: Implemented

Answer: You can upload your running data in GPX or TCX format from your GPS device or smartwatch. Our system will automatically parse the data and extract key information including distance, pace, elevation, heart rate, and route.

Read detailed explanation

Running Data Parsing Steps:

  1. Upload: Click "Upload Data" and select your GPX or TCX file
  2. Parse: Our system uses xml2js to parse the XML data
  3. Extract: Key information is automatically extracted:
    • Trackpoints (latitude, longitude, elevation, time)
    • Lap data (total time, distance, calories, heart rate)
    • Statistics (total distance, elevation gain, average pace, total time)
  4. Review: Check the extracted data and use it for training plan generation

Supported Formats: GPX (GPS Exchange Format), TCX (Training Center XML)

API Endpoint: POST /api/parse-data
